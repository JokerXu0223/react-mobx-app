## react-mobx-app

### Version
```html
node 8.0+(v8.12.0)
npm 5.0+(v6.4.1)
```

### setup
* Run 'yarn setup'

### serve for ios/android
* Run 'yarn ios'
* OR
* Run 'yarn android'


### Project structure
```html
|--src              Source dir
  |--assets         Static source
    |--img          
    |--fonts        maybe you need             
  |--components     Only common & Layout
  |--constants      
  |--models         Mobx store
  |--pages          Container & Self component
  |--services       Xhr api
  |--utils          
  |--index.js       Entry of Program

```
